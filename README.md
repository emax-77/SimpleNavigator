# Simple navigation app for Android created in .NET MAUI
## Set your destination and follow an intuitive compass that points you in the right direction.
### Perfect for use cases like:
- hiking and geocaching
- locating your parked car in a huge parking lot :)

### What's new
- multilanguage support (English-Slovak)
- location management

### Works offline - no internet access required. You need to allow GPS only.
